# Character Performance

Use this file whenever people appear. Write characters as actors with body weight, emotion, and environmental response.

## Core Dimensions

Include the relevant items:

- posture
- gait / rhythm / body weight
- head direction and gaze
- facial expression or visible emotion
- hand behavior
- breathing
- clothing motion
- hair motion
- interaction with props, environment, or other people
- identity and wardrobe consistency

## Back-Follow Character Template

```text
角色背脊挺直，步伐缓慢稳定、有真实重量，肩膀微微放松，头部略微抬起看向远处[目标]。右手轻轻压住被风吹起的衣袖，左手自然垂落，手指随步伐轻微摆动。多数时间只看到背影和侧后方轮廓，偶尔转头露出半侧脸，神情沉静、敬畏、克制。呼吸带来轻微胸口起伏，裙摆和发丝随风自然摆动，人物服装颜色、发型、身形全程一致。
```

## Expression Bank

Use restraint for realism:

- calm and awed: `神情沉静、敬畏、克制`
- tense: `眼神紧张但压抑，嘴唇微微收紧`
- relieved: `呼吸慢慢放松，眼神变柔`
- suspicious: `视线短暂停顿，眉头轻微收紧`
- shy: `眼神短暂躲闪，耳尖微红，嘴角轻微上扬`
- heroic: `目光坚定，下颌微收，步伐沉稳`

Avoid exaggerated expressions unless cartoon, comedy, or melodrama is requested.

## Gait And Body Weight

- `步伐缓慢稳定、有真实重量`: grounded, not floating.
- `脚步轻但不飘`: fantasy elegance while preserving physics.
- `步频均匀`: stable walk cycle.
- `身体随步伐轻微起伏`: realism.
- `转身时衣摆有延迟`: cloth follows body naturally.
- `脚底与地面接触稳定`: reduces sliding.

## Hands

Hands often fail, so keep them simple:

- `双手自然垂落，手指放松`
- `右手轻扶衣袖，左手自然摆动`
- `手掌轻触栏杆但不抓握变形`
- `手指动作小而清晰`
- `避免复杂手势，避免手指变形`

## Cloth, Hair, And Environment

Use physics language:

- `衣摆随步伐和高空风自然摆动`
- `发丝被侧风轻轻吹起`
- `袖口和裙摆有轻微延迟运动`
- `布料有真实重量，不像漂浮丝带`
- `雾气经过人物身侧被轻微扰动`

## Passersby

Give background people small, believable actions:

```text
路人动作自然克制：有人低头提灯走过，有人侧身让路，有人在远处停步仰望宫殿。路人不看镜头，不抢主体，不突然贴近主角，行走方向与空间透视一致。
```

## Dialogue / Lip Sync

When dialogue appears:

- Quote exact lines.
- Add role and emotion.
- Request restrained mouth movement.
- State no subtitles if subtitles are unwanted.

```text
台词（女主，低声克制）："这里，就是天门。" 口型自然同步，说话时嘴部动作小而真实，不夸张，不出现字幕。
```

## Stability Clauses

Add when identity matters:

- `人物脸型、发型、服装颜色、身形比例全程一致`
- `不要变脸、不要换衣、不要多出手指、不要肢体漂移`
- `背影和半侧脸保持同一人`
- `路人与主角保持自然距离，不发生穿模或碰撞`
